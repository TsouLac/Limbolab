# Limbolab
This Github project contains all the code and data used to write the report on the Schiffnen-Morat connection project for the "Aquatic ecosystems: Glaciers, rivers and Lakes" course.

Datas:

1. Hydrological impact
- Daily_dataset_hydro_Murtensee_2016 : provides daily weather and inflow-outflow and lake level data for Murtensee for the year 2016.
- hourly_dataset_hydro_Murtensee_Schiffensee.csv: provides hourly weather and inflow-outflow and lake level data for Murtensee for the year 2016.

2. Biological impact
2. a. Phosphorus
- P_inflow_outflow_Murten : annual water discharges (m3/s) and average TP and PO4 concentrations (mg/m3) in lake Morat’s inflows and outflow (La Broye Canal) between 1988 and 2008. 
- Murtensee_P: annual average concentrations in TP and PO4 (winter average) and in the epilimnion in mg/l in lake Morat 1988-2024.
- P_sediment fluxes : estimated annual sediment burial in total phosphorus (in tP/year).
